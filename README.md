# One Pace English Dub Subtitles

A work-in-progress project to create missing English dub subtitles for [One Pace](https://onepace.net), a fan-edited version of the One Piece anime.

## Tools & Methods

- **[Whisper X](https://github.com/m-bain/whisperX)** - Automated speech-to-text transcription (with modifications)
- **[fun-pace-subs](https://github.com/ang3lo-azevedo/fun-pace-subs)** - Referenced implementation (with extensive modifications)
- **Manual Review** - Quality assurance and correction of generated subtitles

## Progress & Methodology

This project is created while actively watching the series, so subtitles are being generated progressively as episodes are watched. Due to this approach:

- There may be gaps at the start of the series
- Subtitles will be added incrementally as the series progresses
- Timings may not be perfect in all cases

## Status

🚧 **Work in Progress** - Subtitles are being actively created and refined as the series is watched.
